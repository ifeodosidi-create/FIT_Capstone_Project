# FIT Capstone Project

Учебный проект для управления бронированиями номеров в отеле.  
Веб‑приложение на **Python (Flask)**, база данных — **PostgreSQL**.  
Для анализа используются **pandas**, **matplotlib** и **seaborn**.

---

##  Возможности

### Клиент
- Просмотр списка номеров.
- Расчёт стоимости брони:
  - завтрак включён;
  - обед/ужин — доплата за каждого гостя;
  - скидка 5% при брони от 3 суток.
- Создание брони с выбором категории и услуг.
- Проверка занятости номера.
- Оплата брони (мок‑эквайринг: кнопка «Подтвердить платёж — да/нет»).
- Отмена брони (если до заезда ≥1 день). Если оплачено — сообщение с телефоном для возврата средств.

### Администратор
- Просмотр всех данных (брони, платежи, транзакции).
- Экспорт данных в **JSON** и **CSV**.
- Аналитика:
  - доходы по категориям;
  - количество гостей по месяцам;
  - топ‑5 популярных номеров;

## Установка и запуск
```bash
# Клонируем репозиторий
git clone https://github.com/<username>/FIT_Capstone_Project.git
cd FIT_Capstone_Project

# Создаём виртуальное окружение
python -m venv venv
venv\Scripts\activate   # Windows
source venv/bin/activate # Linux/MacOS

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем сервер
python main.py
```
## Откройте в браузере
http://127.0.0.1:5000/
